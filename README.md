## 🎯 What is OISignalFlow?

OISignalFlow is an intelligent command-line options screener that detects **long buildup signals** in NSE FNO stocks:

- 📊 **Real-time Analysis** — Scans up to 209 FNO stocks every 5 minutes
- 🟢 **CE Buy Signals** — Identifies high-probability call buying opportunities
- 📈 **OI Buildup Detection** — Combines OI + Price + Volume analysis
- 📱 **Telegram Alerts** — Instant notifications on your phone
- 🖥️ **Live Dashboard** — Professional dark-theme HTML monitoring dashboard
- 💾 **Excel Export** — All results saved automatically
- ⏰ **Market Hours Only** — Runs 9:15 AM – 3:30 PM IST (Mon–Fri)
- 🆓 **Zero Cost** — Free NSE data via nselib (no paid APIs)

### Signal Types

| Signal | Meaning | Action |
|--------|---------|--------|
| 🟢 **LONG BUILDUP** | Price ↑ + OI ↑ | ✅ **BUY CE** (Best) |
| 🔴 **SHORT BUILDUP** | Price ↓ + OI ↑ | ⚠️ Consider PE |
| 🟡 **SHORT COVERING** | Price ↑ + OI ↓ | ⚠️ Weak signal |
| ⚪ **LONG UNWINDING** | Price ↓ + OI ↓ | ❌ Avoid |

## 🚀 Quick Start

### Prerequisites
- Python 3.7+
- A modern browser (Chrome, Firefox, Edge)
- Telegram account (for alerts)

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Get Telegram Credentials
- **Token** → Chat with [@BotFather](https://t.me/BotFather) → `/newbot`
- **Chat ID** → Chat with [@userinfobot](https://t.me/userinfobot) → `/start`

### 3. Configure `.env`
```bash
cp .env.example .env
```
Edit `.env`:
```ini
TELEGRAM_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
TELEGRAM_CHAT_ID=987654321

OI_CHANGE_MIN=2.0
PRICE_CHANGE_MIN=0.3
VOLUME_MULT=1.5
SCAN_INTERVAL=5
```

### 4. Start the Scanner
```bash
python OISignalFlow.py
```

### 5. Open the Dashboard
```bash
# Option A — direct file open (works in all browsers)
open index.html

# Option B — via local server (recommended for live fetch)
python3 -m http.server 8000
# Then visit: http://localhost:8000
```

## 📋 Configuration

All settings managed via `.env`:

| Variable | Default | Description |
|----------|---------|-------------|
| `TELEGRAM_TOKEN` | — | Bot token from @BotFather |
| `TELEGRAM_CHAT_ID` | — | Your Telegram user ID |
| `OI_CHANGE_MIN` | 2.0 | Min OI change % for alert |
| `PRICE_CHANGE_MIN` | 0.3 | Min price change % for alert |
| `VOLUME_MULT` | 1.5 | Volume must be X times normal |
| `SCAN_INTERVAL` | 5 | Scan frequency (minutes) |
| `REQUEST_DELAY` | 0.3 | Seconds between NSE API requests (rate limiting) |
| `ENABLE_ADX_FILTER` | False | Enable ADX trend strength filter |
| `ADX_MIN` | 25.0 | Minimum ADX value required (25 = strong trend) |
| `ADX_PERIOD` | 14 | ADX calculation period (14 = industry standard) |
| `OUTPUT_FILE` | OISignalFlow_Results.xlsx | Excel output filename |
| `LOG_LEVEL` | INFO | Logging level (DEBUG/INFO/WARNING/ERROR) |
| `ENABLE_DESKTOP_NOTIFICATIONS` | True | Desktop notification on/off (set False on server) |

## 🐛 Troubleshooting

| Problem | Fix |
|---------|-----|
| `ModuleNotFoundError: dotenv` | `pip install python-dotenv` |
| Telegram: Unauthorized | Check `TELEGRAM_TOKEN` in `.env` |
| Telegram: Bad Request | Check `TELEGRAM_CHAT_ID` is numbers only |
| No alerts during market hours | Lower `OI_CHANGE_MIN` in `.env` |
| Dashboard shows "No data" | Make sure `OISignalFlow.py` is running |
| Dashboard not updating | Open via `python3 -m http.server 8000` instead of direct file |
